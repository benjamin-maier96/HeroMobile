# Hero Mobile
An ARPG idle game, that provides deep build functionality like a POE2, but with a small time investment.
## Target group
The game should target player that want to discover deep game mechanics and combos like in an arpg, but without the constant grind.
## Core Loop
A player starts a run by selecting a class. During a run a player gains experiences and levels up. Here the player can pick different skills and modifiers. He also loots items,
which will be available between runs.
### Open questions
- When does a run end? It could end because the player could not progress any further without obtaining new prestige modifier. Another option would be to add some kind of age
  to the character, so he has only a limited amount of actions. However, this would then probably speak against an idle game.
